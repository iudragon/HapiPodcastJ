HapiPodcastJ

This is Hapi Podcast J, a fork of Hapi Podcast, an Android app for playing podcasts.

Hapi Podcast and Hapi Podcast J are open source software released under the GPL. The git repository includes the full history that is available in the Hapi Podcast svn repository.

It has a few new features and a rearranged UI.
